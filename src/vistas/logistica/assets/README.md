# Assets para Órdenes de Compra

Esta carpeta contiene las imágenes necesarias para generar las órdenes de compra en PDF.

## Imágenes requeridas:

1. **logo-hpk.png** - Logo de HP&K INVERSIONES S.R.L.
   - Dimensiones recomendadas: 200x60 px
   - Formato: PNG con fondo transparente

2. **firma-diego-jaime.png** - Firma de Diego Jaime (Elaborado por)
   - Dimensiones recomendadas: 150x60 px
   - Formato: PNG con fondo transparente
   - Incluye firma manuscrita y sello de HP&K INVERSIONES SRL

3. **firma-diego-munoz.png** - Firma de Diego Muñoz (Revisado por)
   - Dimensiones recomendadas: 150x60 px
   - Formato: PNG con fondo transparente
   - Incluye firma manuscrita y sello con "Sup de Operaciones"

## Instrucciones:

1. Guarda las 3 imágenes proporcionadas en esta carpeta
2. Asegúrate de que los nombres coincidan exactamente con los listados arriba
3. Las imágenes se cargarán automáticamente en el PDF de las órdenes de compra
