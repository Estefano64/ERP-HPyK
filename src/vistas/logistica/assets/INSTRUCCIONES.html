📋 Configuración de Imágenes para Órdenes de Compra

Para que las órdenes de compra se generen con el formato profesional de HP&K, necesitas agregar 3 imágenes en formato PNG en esta carpeta.

🖼️ Imágenes Requeridas

1️⃣ Logo de HP&K

Nombre del archivo: logo-hpk.png

Descripción: Logo de HP&K INVERSIONES S.R.L. (azul y celeste)

Uso: Aparece en la esquina superior izquierda del PDF

Dimensiones recomendadas: 200 x 60 píxeles

2️⃣ Firma de Diego Jaime

Nombre del archivo: firma-diego-jaime.png

Descripción: Firma manuscrita con sello "HP&K INVERSIONES SRL - ELABORADO POR"

Uso: Aparece en el cuadro "ELABORADO POR"

Dimensiones recomendadas: 150 x 60 píxeles

3️⃣ Firma de Diego Muñoz

Nombre del archivo: firma-diego-munoz.png

Descripción: Firma manuscrita con sello "HP&K INVERSIONES SRL - Sup de Operaciones"

Uso: Aparece en el cuadro "REVISADO POR"

Dimensiones recomendadas: 150 x 60 píxeles

📝 Instrucciones Paso a Paso

⚠️ IMPORTANTE: Los nombres de los archivos deben ser EXACTAMENTE como se indica (respetando mayúsculas y minúsculas).
  1. Guarda las 3 imágenes que te proporcionaron en esta carpeta (assets)
  2. Asegúrate de que los nombres sean:
    • logo-hpk.png
    • firma-diego-jaime.png
    • firma-diego-munoz.png
  3. Refresca la página del ERP en tu navegador (F5)
  4. Ve a Compras → Órdenes de Compra
  5. Haz clic en el botón de PDF (icono morado) de cualquier orden

✅ Verificación

✓ Si las imágenes se cargaron correctamente: Verás en la consola del navegador (F12) el mensaje:

Imágenes OC cargadas: { logo: 'OK', firmaDiegoJaime: 'OK', firmaDiegoMunoz: 'OK' }

🔍 ¿Cómo verificar?

  1. Abre el navegador
  2. Presiona F12 para abrir las herramientas de desarrollo
  3. Ve a la pestaña "Console"
  4. Revisa los mensajes de carga de imágenes

🚨 Solución de Problemas

Si las imágenes no aparecen en el PDF:

  • Verifica que los nombres de los archivos sean exactos
  • Asegúrate de que las imágenes estén en formato PNG
  • Refresca la página del ERP (F5) después de copiar las imágenes
  • Verifica que las imágenes no estén dañadas abriendo con un visor de imágenes

📌 Notas Adicionales

  • Si las imágenes NO están disponibles, el PDF se generará igual pero con texto en lugar de las imágenes
  • Las imágenes se cargan automáticamente al abrir la sección de Órdenes de Compra
  • El formato PNG con fondo transparente es recomendado para mejor visualización
  • Las dimensiones son flexibles pero se ajustarán automáticamente en el PDF
